## 赛事名称

清华大学自动化系第二届具身智能挑战赛

## 赛事主题

月球救援

## 赛事背景

<center><strong>登陆月球，展开一场与时间赛跑的救援！</strong></center>

在「月球救援」主题下，参赛队伍需要为智能小车编写程序，让它在模拟月面场景中自主感知、导航并完成一系列救援任务，全面考验图像识别、避障导航、机械臂操作与多任务协调能力。相关的硬件搭建与技术模块教程（小车安装、开发工具、SSH、摄像头与超声波、麦轮控制、机械臂控制、二维码识别等）均收录在平台仓库中。
